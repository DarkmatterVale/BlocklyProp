h2(#servo). Servos

Servos can be used using BlocklyProp's Spin compiler. Below are all of the available servo blocks that you have access to. Hope this helps!

h2(#servo_pulse). Send pulses to a servo

This block, when added will send pulses to a servo. The allowable values for the pulses include anything between 1500 - 2500.
